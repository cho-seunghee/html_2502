지침v-bind​

CSS 규칙이 설정되어 있고 병합되어 있는 것을 볼 수 있는 class 가 있습니다. (class="impClass14" :class="{yelClass14:isYellow}")

'impClass' 클래스는 class 속성으로 일반적인 방식으로 설정되고, 'yelClass' 클래스는 v-bind:class로 설정됩니다. 결과적으로 Vue는 클래스를 병합하여 div 요소가 두 클래스에 모두 속하게 됩니다.

이 div는 'impClass'와 'yelClass' 둘 다에 속합니다.

지침v-bind​

여러 개의 클래스를 v-bind:class 쉼표로 구분하여 설정할 수 있습니다. (:class="{ yelClass15: isYellow, impClass15: isImportant}")

Vue에서는 클래스를 쉼표로 구분하여 여러 클래스를 쉽게 할당할 수 있습니다.

이는 'isYellow' 및 'isImportant' Vue 속성에 따라 'impClass'와 'yelClass' 모두에 속할 수 있습니다.

CSS 속성 이름에는 'v-bind:style'을 사용하여 낙타 표기법과 따옴표로 묶인 케밥 표기법을 모두 사용할 수 있지만, 낙타 표기법을 사용하는 것이 좋습니다.
