translate 변형 함수

이미지 이동하기 

                        -webkit-transform: translateX(50px);
                        -ms-transform: translateX(50px);
                        -moz-transform: translateX(50px);
                        -o-transform: translateX(50px);
                        transform: translateX(50px); //변환(transform) 속성이고, X축 방향으로 50픽셀 만큼 오른쪽으로 이동

                        transform: translateY(20px); //변환(transform) 속성이고, Y축 방향으로 20픽셀 만큼 오른쪽으로 이동

                        transform: translate(10px, 20px); //변환(transform) 속성이고, X축 방향으로 10픽셀 만큼 Y축 방향으로 20픽셀 만큼 오른쪽으로 이동

sacle 변형 함수 - 요소 확대/축소하기

이미지 확대/축소하기 

                        -webkit-transform: scaleX(1.2);
                        -ms-transform: scaleX(1.2);

rotate 변형 함수 - 요소 회전하기

이미지 2차원 회전하기 

                        //단위: deg(도), rad(라디안), turn(회전) 사용 가능
                        -webkit-transform: rotate(20deg);
                        -ms-transform: rotate(20deg);
                        -moz-transform: rotate(20deg);
                        -o-transform: rotate(20deg);
                        transform: rotate(20deg); //20도 시계 방향 회전

                        transform: rotate(-40deg); //40도 반시계 방향 회전

rotate 변형 함수 - 요소 회전하기

이미지 3차원 회전하기 

                        -webkit-transform: rotateX(45deg);
                        -moz-transform: rotateX(45deg);
                        -o-transform: rotateX(45deg);
                        -ms-transform: rotateX(45deg);
                        transform: rotateX(45deg); //X축 기준으로 45도 회전 (3D 회전)

                        transform: rotateY(45deg); //Y축 기준으로 45도 회전 (3D 회전)

                        transform: rotateZ(45deg); //Z축 기준으로 45도 회전 (3D 회전)

                        transform: rotate3d(2.5, 1.2, -1.5, 45deg); X축, Y축, Z축, 회전각도 회전(3D 회전)

skew 변형 함수 - 요소를 비틀어 왜곡하기

x축과 y축으로 이미지 왜곡하기 

                        -webkit-transform: skewX(30deg);
                        -ms-transform: skewX(30deg);
                        -moz-transform: skewX(30deg);
                        -o-transform: skewX(30deg);
                        transform: skewX(30deg); //X축 기준으로 30도 기울림

                        transform: skewY(15deg); //Y축 기준으로 15도 기울림

                        transform: skew(-25deg, -15deg); //X축 -25도, Y축 -15 기울림

transform-origin 속성 - 변형 기준점 설정하기

이미지 변형 시작점을 다양하게 지정하기 

                        //transform-origin 속성 : 요소의 변환(transform)이 일어나는 기준점
                        -moz-transform: rotateZ(10deg);
                        -ms-transform: rotateZ(10deg);
                        -o-transform: rotateZ(10deg);
                        -webkit-transform: rotateZ(10deg);
                        transform: rotateZ(10deg); //Z축 기준으로 10도 회전

                        transform-origin: left top; //중심점을 요소의 왼쪽 상단으로 설정

                        transform-origin: right top; //중심점을 요소의 오른쪽 상단으로 설정

                        transform-origin: left bottom; //중심점을 요소의 왼쪽 하단으로 설정

                        transform-origin: right bottom; //중심점을 요소의 오른쪽 하단으로 설정

perspective, perspective-origin 속성 - 원근감 표현하기

이미지 회전하며 원근감 주기 

                        //perspective: 3D 공간에서 관찰자(사용자)와 평면 사이의 거리를 정의(3D 변환에서 원근감을 부여)
                        transform: rotateX(50deg);

                        perspective: 300px; //300픽셀만큼 원근감 (값이 낮을 수록 원근감이 강해짐)

transform-style 속성 - 3D 변형 적용하기

3D 변형을 적용하지 않았을 때 vs 적용했을 때 

                        //transform-style 속성 : 요소의 자식 요소들이 3D 공간에서 어떻게 렌더링될지를 결정
                        transform: rotateY(45deg);

backface-visibility 속성 - 요소의 뒷면 표시하기

요소의 뒷면을 표시하지 않을 때 vs 표시할 때 

                        //backface-visibility 속성 :요소의 뒷면이 사용자를 향할 때 해당 요소를 보이게 할지 숨길지 결정

@keyframes 속성 - 애니메이션 지점 설정하기

사각형에서 원으로 서서히 바뀌는 애니메이션 

                        //animation-duration : CSS 애니메이션의 한 주기가 완료되는 데 걸리는 시간을 지정하는 속성
                        //@keyframes change-bg는 CSS 애니메이션을 정의하는 규칙으로, 배경색 변경과 같은 애니메이션 시퀀스를 만드는 데 사용
                        animation-duration: 3s; //시간이 3초동안 걸림
                        
                        @keyframes change-bg {
                            from {
                                background-color: blue;
                                border: 1px solid black;
                            }
                            to {
                                background-color: #a5d6ff;
                                border:1px solid blue;
                                border-radius: 50%;
                            }
                        }

animation-duration 속성 - 애니메이션 실행 시간 설정하기

animation-direction 속성 - 애니메이션 방향 지정하기

animation-iteration-count 속성 - 반복 횟수 지정하기

무한 반복하는 애니메이션 만들기 

                        //animation-direction: CSS 애니메이션의 재생 방향을 지정하는 속성
                        -normal: 기본값, 정방향으로 재생
                        -reverse: 역방향으로 재생
                        -alternate: 정방향과 역방향을 번갈아 재생
                        -alternate-reverse: alternate의 반대로 시작

                        //animation-iteration-count : CSS 애니메이션의 반복 횟수를 지정하는 속성
                        -infinite: 무한 반복
                        -숫자 값: 특정 횟수만큼 반복 (예: 1, 2, 2.5 등)
                        -기본값: 1

                        animation-name: moving;
                        animation-duration: 3s;
                        animation-direction: normal;
                        animation-iteration-count: infinite;

                        @keyframes moving {
                            from {
                                width: 200px;
                                background: #faef7c;
                                opacity: 0.5;
                                transform: scale(0.5) rotate(15deg);
                            }
                        
                            to {
                                width: 400px;
                                background: #ff9400;
                                opacity: 1;
                                transform: scale(1) rotate(0deg);
                            }
                        }

                        animation: rotate 1.5s infinite, background 1.5s infinite alternate;
                        //이름: rotate
                        //지속시간: 1.5초
                        //반복: 무한(infinite)
                        //이름: background
                        //지속시간: 1.5초
                        //반복: 무한(infinite)
                        //방향: alternate (정방향/역방향 번갈아가며)

                        @keyframes rotate {
                            from {
                                transform: perspective(120px) rotateX(0deg) rotateY(0deg);
                            }
                            50% {
                                transform: perspective(120px) rotateX(-180deg) rotateY(0deg);
                            }
                            to {
                                transform: perspective(120px) rotateX(-180deg) rotateY(-180deg);
                            }
                        }

                        @keyframes background {
                            from {
                                background: red;
                            }
                        
                            50% {
                                background-color: green
                            }
                        
                            to {
                                background-color: blue;
                            }
                        }
