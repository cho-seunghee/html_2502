transform과 변형 함수

translate 변형 함수

이미지 이동하기 

                        -webkit-transform: translateX(50px);
                        -ms-transform: translateX(50px);
                        -moz-transform: translateX(50px);
                        -o-transform: translateX(50px);
                        transform: translateX(50px); //X축 방향으로 50픽셀 만큼 오른쪽으로 이동

                        transform: translateY(20px);

                        transform: translate(10px, 20px);

CSS와 애니메이션

@keyframes 속성 - 애니메이션 지점 설정하기

사각형에서 원으로 서서히 바뀌는 애니메이션 

                        animation-duration: 3s;
                        
                        @keyframes change-bg {
                            from {
                                background-color: blue;
                                border: 1px solid black;
                            }
                            to {
                                background-color: #a5d6ff;
                                border:1px solid blue;
                                border-radius: 50%;
                            }
                        }

animation-name 속성 - 애니메이션 이름 지정하기

@keyframe을 사용해 실행할 애니메이션 지정하기 

                        animation-name: shape;
                        animation-duration: 3s;

                        animation-name: rotate;
                        animation-duration: 3s;

                        @keyframes shape {
                            from {
                                border: 1px solid black;
                            }
                        
                            to {

                                border-radius: 50%;
                            }
                        }

                        @keyframes rotate {
                            from {
                                transform:rotate(0deg)
                            }
                        
                            to {
                                transform: rotate(45deg);
                            }
                        }

animation-duration 속성 - 애니메이션 실행 시간 설정하기

animation-direction 속성 - 애니메이션 방향 지정하기

animation-iteration-count 속성 - 반복 횟수 지정하기

무한 반복하는 애니메이션 만들기

                        animation-duration: 3s;
