CSS 테두리 너비

The border-width Property

This property specifies the width of the four borders:

border-style: solid;border-width: 5px;

border-style: solid;border-width: medium;

border-style: dotted;border-width: 2px;

border-style: dotted;border-width: thick;

border-style: double;border-width: 15px;

border-style: double;border-width: thick;

Note: The "border-width" property does not work if it is used alone. Always specify the "border-style" property to set the borders first.

특정 측면 너비

The border-width Property

The border-width property can have from one to four values (for the top border, right border, bottom border, and the left border):

border-style: solid;border-width: 5px 20px;

border-style: solid;border-width: 20px 5px;

border-style: solid;border-width: 25px 10px 4px 35px;

CSS 테두리 색상

The border-color Property

This property specifies the color of the four borders:

border-style: solid;border-color: red;

border-style: solid;border-color: green;

border-style: dotted;border-color: blue;

Note: The "border-color" property does not work if it is used alone. Use the "border-style" property to set the borders first.

특정 측면 색상

The border-color Property

The border-color property can have from one to four values (for the top border, right border, bottom border, and the left border):

border-style: solid;border-color: red green blue yellow;

HEX 값

The border-color Property

The color of the border can also be specified using a hexadecimal value (HEX):

border-style: solid;border-color: #ff0000;

border-style: solid;border-color: #0000ff;

border-style: solid;border-color: #bbbbbb;

RGB 값

The border-color Property

The color of the border can also be specified using RGB values:

border-style: solid;border-color: rgb(255, 0, 0);

border-style: solid;border-color: rgb(0, 0, 255);

border-style: solid;border-color: rgb(187, 187, 187);

HSL 값

The border-color Property

The color of the border can also be specified using HSL values:

border-style: solid;border-color: hsl(0, 100%, 50%);

border-style: solid;border-color: hsl(240, 100%, 50%);

border-style: solid;border-color: hsl(0, 0%, 73%);

CSS 둥근 테두리

The border-radius Property

This property is used to add rounded borders to an element:

border: 2px solid red;padding: 5px;

border: 2px solid red;border-radius: 5px;padding: 5px;

border: 2px solid red;border-radius: 8px;padding: 5px;

border: 2px solid red;border-radius: 12px;padding: 5px;
