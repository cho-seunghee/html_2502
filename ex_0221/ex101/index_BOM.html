브라우저 쿠키

네비게이터 객체

navigator.cookieEnabled

cookieEnabled 속성은 쿠키가 활성화되어 있는 경우 true를 반환합니다.:

브라우저 애플리케이션 이름

네비게이터 객체

navigator.appName

appName 속성은 브라우저의 애플리케이션 이름을 반환합니다.:

이상하게도 "Netscape"는 IE11, Chrome, Firefox, Safari의 애플리케이션 이름입니다.

브라우저 응용 프로그램 코드 이름

네비게이터 객체

navigator.appCodeName

appCodeName 속성은 브라우저의 코드 이름을 반환합니다.

그것에 의존하지 마세요! "Mozilla"는 Chrome, Firefox, IE, Safari, Opera의 애플리케이션 코드명입니다.

브라우저 엔진

네비게이터 객체

navigator.product

product 속성은 브라우저의 제품 이름을 반환합니다.

그것에 의존하지 마세요! 대부분 브라우저는 제품 이름으로 "Gecko"를 반환합니다!

브라우저 버전

네비게이터 객체

navigator.appVersion

appVersion 속성은 브라우저에 대한 버전 정보를 반환합니다.:

navigator.userAgent

userAgent 속성은 브라우저에서 서버로 보낸 사용자 에이전트 헤더를 반환합니다.:
